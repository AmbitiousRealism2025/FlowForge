# PWA Icons

Place your PWA icon files here with the following sizes:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

You can generate these from a single source icon (1024x1024 recommended) using tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

The icons should feature the FlowForge brand with the flow-green (#00D9A5) color scheme.
